## Visão geral

O Shadow Whisper é uma ferramenta de voz para texto. Processamos seu áudio para fornecer transcrição e melhoria de texto. Esta política explica quais dados coletamos, por quê, por quanto tempo os mantemos e seus direitos.

Estamos comprometidos com os princípios de privacidade desde o design e minimização de dados. Coletamos apenas o necessário para fornecer o serviço.

## Dados que coletamos

### Dados da conta

Quando você cria uma conta, coletamos:

| Dado                   | Finalidade                            | Retenção                |
| ---------------------- | ------------------------------------- | ----------------------- |
| Endereço de email      | Autenticação e identificação da conta | Até exclusão da conta   |
| Nome (se fornecido)    | Exibição no aplicativo                | Até exclusão da conta   |
| Tokens de autenticação | Gerenciamento de sessão               | Até logout ou expiração |

**Base legal:** Execução de contrato — necessário para fornecer o serviço contratado.

### Dados de áudio

Quando você usa o Shadow Whisper para transcrever:

- Seu áudio é **enviado para nosso serviço na nuvem** para transcrição via IA de fala para texto.
- O áudio é processado em tempo real e **não é armazenado em nossos servidores** após a conclusão da transcrição.
- A extensão do navegador **não** salva arquivos de áudio localmente.

**Base legal:** Execução de contrato — a transcrição é o serviço principal.

### Dados de transcrição

- O **texto transcrito** (bruto e limpo) é armazenado **localmente no seu navegador** via armazenamento local da extensão.
- As transcrições **não são enviadas ou armazenadas em nossos servidores** além da sessão de processamento.
- O histórico local pode ser apagado a qualquer momento nas configurações da extensão.

**Base legal:** Execução de contrato — o histórico local é uma funcionalidade solicitada.

### Análise de uso

Coletamos estatísticas de uso anonimizadas para entender como o produto é utilizado:

| Dado                            | Finalidade                             |
| ------------------------------- | -------------------------------------- |
| Contagem de palavras por sessão | Estatísticas pessoais de produtividade |
| Duração da sessão               | Estatísticas pessoais de produtividade |
| Domínio do app/site ativo       | Formatação de texto contextual         |
| Idioma detectado                | Qualidade do serviço                   |
| Fuso horário                    | Exibição e cálculo de sequências       |

**Não** coletamos:

- Títulos de páginas ou nomes de abas
- Conteúdo de páginas ou texto de acessibilidade
- URLs além do nome de domínio
- Histórico de navegação

**Base legal:** Execução de contrato (estatísticas pessoais) e legítimo interesse (qualidade do serviço), com minimização de dados aplicada.

### Dados de cobrança

Se você assinar um plano pago:

| Dado                  | Finalidade                           | Processado por          |
| --------------------- | ------------------------------------ | ----------------------- |
| Método de pagamento   | Cobrança de assinatura               | Stripe                  |
| Histórico de cobrança | Geração de faturas e recibos         | Stripe                  |
| Status da assinatura  | Controle de acesso a funcionalidades | Shadow Whisper + Stripe |

Não armazenamos números de cartão de crédito ou detalhes de pagamento diretamente. Todo o processamento de pagamentos é feito pelo **Stripe**.

**Base legal:** Execução de contrato e obrigações legais/regulatórias.

### Dicionário e skills

Suas entradas de dicionário pessoal, snippets e configurações de skills são armazenadas em nossos servidores para sincronização entre sessões.

**Base legal:** Execução de contrato.

## Dados que NÃO coletamos

- **Títulos de páginas, nomes de abas ou títulos de janelas** — removidos por privacidade.
- **Conteúdo de tela ou texto de acessibilidade** — nunca coletados.
- **Histórico de navegação ou atividade** — não monitorado.
- **Gravações de áudio** — não retidas após a transcrição.
- **Conteúdo de transcrição em nossos servidores** — não armazenado além da sessão de processamento.

## Serviços terceiros (suboperadores)

Utilizamos os seguintes serviços para operar o Shadow Whisper:

| Serviço          | Finalidade                                        | Dados compartilhados                              | Localização |
| ---------------- | ------------------------------------------------- | ------------------------------------------------- | ----------- |
| **Cloudflare**   | Hospedagem, CDN, Workers (computação), AI Gateway | Metadados de requisição, áudio (para transcrição) | Global      |
| **Stripe**       | Processamento de pagamentos                       | Dados de cobrança                                 | Global      |
| **Google OAuth** | Autenticação (se escolhido)                       | Email, nome                                       | Global      |

Todos os suboperadores estão vinculados por contratos de processamento de dados.

## Transferência internacional de dados

O Shadow Whisper é operado a partir do Brasil. Alguns dados podem ser processados fora do Brasil através de nossos suboperadores (Cloudflare, Stripe, Google). Essas transferências se baseiam nas cláusulas contratuais padrão e certificações de proteção de dados dos suboperadores como mecanismos previstos no Artigo 33 da LGPD.

## Retenção de dados

| Tipo de dado             | Retenção                                                             |
| ------------------------ | -------------------------------------------------------------------- |
| Dados da conta           | Até exclusão da conta                                                |
| Áudio                    | Não retido após transcrição                                          |
| Histórico de transcrição | Apenas local, controlado por você                                    |
| Análise de uso           | Agregado, retido para operação do serviço                            |
| Dados de cobrança        | Conforme exigido por lei (geralmente 5 anos para obrigações fiscais) |
| Dicionário e skills      | Até exclusão pelo usuário ou exclusão da conta                       |

## Seus direitos

Sob a Lei Geral de Proteção de Dados (LGPD) e regulamentações de privacidade aplicáveis, você tem o direito de:

- **Acessar** seus dados pessoais.
- **Corrigir** dados imprecisos.
- **Excluir** seus dados (direito à eliminação).
- **Exportar** seus dados em formato portável.
- **Revogar consentimento** quando o processamento for baseado em consentimento.
- **Se opor** ao processamento baseado em legítimo interesse.
- **Solicitar informações** sobre com quais entidades seus dados foram compartilhados.

Para exercer qualquer um desses direitos, entre em contato pelo email [privacy@chainbreakertech.com](mailto:privacy@chainbreakertech.com).

## Dados de crianças

O Shadow Whisper não é direcionado a crianças menores de 13 anos. Não coletamos intencionalmente dados pessoais de crianças menores de 13 anos. Se você acredita que uma criança menor de 13 anos nos forneceu dados pessoais, entre em contato conosco e tomaremos medidas para excluí-los.
