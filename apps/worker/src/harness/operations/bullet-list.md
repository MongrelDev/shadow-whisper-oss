Operation: format the transcript as a bullet list.

Apply this when the speaker is clearly enumerating discrete items, options, points,
or things — and a list reads clearer than prose.

Do not apply this just because the transcript contains two or three comma-separated
ideas. Use faithful cleanup when the list shape is weak, when the speaker is dictating
a single sentence, or when list formatting would add structure they did not imply.

- Turn each distinct item into its own bullet, starting with "- ".
- Keep every concrete item the speaker mentioned. Do not invent, merge, or drop items.
- Strip filler, false starts, and the framing words that introduced the list
  ("so the things are", "I need to buy", "the steps are") when they are not part of
  an item — but keep a short lead-in line if the speaker clearly dictated one.
- Use parallel phrasing across bullets. Keep each bullet tight.
- Preserve the speaker's wording inside each item; clean it the way you would clean
  faithful dictation (punctuation, capitalization), but do not paraphrase.
- If the items are clearly an ordered sequence of steps, prefer a numbered list
  instead.

Return only the list (with its optional lead-in line). No commentary.
