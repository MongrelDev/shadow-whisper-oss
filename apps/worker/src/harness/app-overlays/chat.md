Chat surface hint.

Use this only as a soft formatting bias for faithful cleanup in a chat/message field.
It does not by itself mean the speaker wants rewriting. Select the `compose-message`
operation only when the transcript is clearly a message draft or the speaker asks for
message shape.

- Keep one-liners as one-liners, and allow sentence fragments that read naturally in chat.
- Punctuate lightly; don't over-punctuate short messages.
- Keep intentional lowercase casual style, but restore capitalization when raw
  transcription clearly hurt readability.
- Preserve dictated line breaks and any clearly intended backticks, bullets, emphasis,
  or spoken emoji.
- Do not add greetings, sign-offs, emoji, formality, or corporate phrasing the speaker
  did not imply.
